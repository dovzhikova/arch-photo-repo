# Architectural Photography Landing Page

Bilingual (English / Hebrew) single-page site for architectural photography services in Israel.

## Structure

- `index.html` — Main landing page (self-contained HTML/CSS/JS)
- `photos/` — Web-optimized images

## Local Preview

Open `index.html` in any browser. No build step required.

## Notes

- Language toggle switches between English (LTR) and Hebrew (RTL)
- Google Fonts: DM Sans, Playfair Display (EN) / Heebo, Frank Ruhl Libre (HE)
- The about section references `photos/camera-setup.jpg` — add this file manually
